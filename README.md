# Python3 web-sites bruteforcer
## Installation:
```bash
git clone https://github.com/fukttt/py3bruteforce.git
pip3 install -r requirements.txt
python3 brute.py -m porn
```
OR 
```
flask run
```
##UPDATE 1.4

Just open http://localhost:5000
